Primer grupisanja sadržaja u HTML-u

Demonstracija upotrebe div i span elemenata.

Grupisanje blokova sadržaja

Element <div> koristi se kao kontejner za organizovanje različitih delova stranice. Na primer, možemo imati posebne blokove za zaglavlje, sadržaj i podnožje stranice.

Ovaj blok je takođe napravljen pomoću div elementa kako bi se izdvojila dodatna informacija.

Grupisanje linijskih elemenata

Element <span> koristi se za označavanje određenih delova teksta koji kasnije mogu biti stilizovani pomoću CSS-a.

Na primer: HTML je jezik za označavanje, dok je CSS jezik za stilizovanje web stranica.
